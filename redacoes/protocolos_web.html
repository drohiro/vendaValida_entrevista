HTTP e HTTPS

Tanto o HTTP como o HTTPS são protocolos web utilizados pelo navegador para fazer a requisição de informação para o servidor, que contenha um site, blog, fórum etc. Resumindo, tudo o que você acessa pelo navegador utiliza e depende totalmente desses protocolos.

O HTTP que é a sigla para “HyperText Transfer Protocol” que em português significa "Protocolo de Transferência de Hipertexto". E um protocolo de transferência de sistemas de informação mais utilizado na World Wide Web (também conhecida como internet), fazendo transferência de arquivos HTML entre o servidor e o solicitante. E é por isso que a URL dos sites geralmente começa com HTTP://, assim mostrando o protocolo utilizado pela página.

Já o HTTPS que é a sigla para “Hyper Text Transfer Protocol Secure” que em português significa “Protocolo de Transferência de Hipertexto Seguro” é a junção dos protocolos HTTP e SSL “Secure Sockets Layers” que adiciona uma criptografia ao protocolo padrão assim o tornando mais seguro que o original. Apesar de ser mais seguro o HTTPS não é invulnerável e também pode ser burlado, ou seja, mesma que a página que você confia utiliza desse protocolo não deixe de tomar todas as medidas de precaução para manter os seus dados o mais seguro possível.
