O que é uma API?

API do inglês “Application Programming Interface” em português “Interface de Programação de Aplicações” refere-se a uma forma de integração de sistemas entregando alta segurança, facilidade na troca de dados com outras linguagens de programação e controle de acesso.

Resumindo uma API eh uma ponte que conecta uma aplicação a outra independente das linguagens que essa utilizando, respeitando somente a entrada e saídas de dados requeridos pela aplicação proprietária que irá fornecer os dados, por exemplo o Google Maps que tem uma API que fornece o seu mapa para qualquer um que queira implementar em seu site, ou o Facebook que fornece uma API que permite fazer cadastro e login em outras aplicações sem a necessidade de preencher cadastros.

Como são invisíveis para os usuários o termo API não é conhecido pela maioria das pessoas, que mesmo não as conhecendo utilizam incontáveis API por dia e dependem de muitas coisas que são resultado da utilização desses serviços.

Hoje em dia quase qualquer aplicação utiliza de alguma API e se você e um desenvolvedor, é quase inevitável precisar usufruir de alguma.
